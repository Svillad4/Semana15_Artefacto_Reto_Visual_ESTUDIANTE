# Rúbrica de evaluación

| Criterio | Puntaje |
|---|---:|
| Producto ejecutable: el proyecto corre siguiendo el README | 25 |
| Organización de carpetas y archivos | 15 |
| README claro, breve y funcional | 15 |
| Evidencia de pruebas con comandos y resultados | 15 |
| Recurso visual pertinente y comprensible | 15 |
| Coherencia con el problema contextual | 10 |
| Registro de roles o aportes del equipo | 5 |
| **Total** | **100** |

## Nivel alto

El proyecto se ejecuta sin errores, genera reporte y gráfico, presenta evidencias claras, registra roles y comunica los resultados con un recurso visual pertinente.

## Nivel medio

El proyecto funciona parcialmente o requiere ajustes menores. La documentación existe pero puede ser más clara.

## Nivel bajo

El proyecto no ejecuta, no genera resultados o carece de evidencias suficientes.
